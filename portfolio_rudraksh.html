My Projects

Project 1

This project transforms the classic Flappy Bird game by using AI and OpenCV to enable hand gesture control. Instead of traditional controls, players use a webcam to raise or lower their hand, which AI interprets in real-time to adjust the bird's altitude. Key features include smooth gesture recognition, AI-powered control, and customizable sensitivity for a tailored experience. OpenCV handles real-time hand detection, while Python and Pygame manage game logic and visuals. The process involves capturing hand movements via webcam, detecting and interpreting gestures with AI, and translating them into in-game bird movements. This innovative project showcases the potential of AI and computer vision for creating immersive, hands-free gaming experiences.

Project 3

This simple book-buying website demonstrates e-commerce fundamentals. It features **book listings** with titles, prices, and a "Buy Now" button. When a user clicks "Buy Now," a **dynamic payment form** appears, prefilled with the book’s title and price. The form includes fields for **card number**, **expiry date**, and **CVC**, with **payment validation** ensuring all fields are correctly filled before proceeding. Upon successful payment, a **confirmation message** appears, thanking the user and confirming the transaction. This project highlights user interface design, form validation, and dynamic JavaScript functionality..
